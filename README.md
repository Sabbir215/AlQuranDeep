### **Common Issues & Solutions**

#### **Audio Not Playing**

- Check your internet connection
- Ensure browser audio is enabled
- Try refreshing the page
- Some verses may not have audio available

- Refresh the page
- Check if the selected language/tafsir combination is available

- Update your mobile browser
