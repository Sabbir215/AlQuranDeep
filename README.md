## 🚀 Quick Start Guide

### **Step 1: Open the Application**

Simply open `index.html` in any modern web browser. No installation required!

### **Step 2: Start Reading**

- The app opens with **Al-Fatiha (Chapter 1, Verse 1)** by default
- You'll see the Arabic text, transliteration, and word-by-word breakdown

### **Step 3: Explore Tafsir (Optional)**

- Click the **"Show"** button in the Tafsir section
- Choose your preferred language and scholarly interpretation

### **Step 4: Navigate**

- Use the **← →** arrows to move between verses
- Use the search box to jump to any verse (e.g., "2-255" for Ayatul Kursi)

That's it! You're ready to explore the Quran. 🎉

## 📚 Complete User Guide

### 🔍 **Navigation Made Simple**

#### **Quick Navigation**

- **← → Arrow Buttons**: Move between verses
- **Search Box**: Type surah name or verse reference
  - Examples: "Fatiha", "Baqarah", "2-255", "18-10"
- **Auto-complete**: Start typing for instant suggestions

#### **Smart Search Examples**

```
"Fatiha"     → Takes you to Al-Fatiha (Chapter 1)
"2-255"      → Jumps to Ayatul Kursi (Chapter 2, Verse 255)
"Kahf"       → Opens Surah Al-Kahf (Chapter 18)
"18-10"      → Chapter 18, Verse 10
```

#### **Ruku Navigation**

- **Jump by 3 verses**: Use Ruku buttons for faster navigation through longer chapters

### 📖 **Understanding the Layout**

#### **Main Content Area**

1. **Surah & Verse Info**: Always visible at the top
2. **Arabic Text**: Original Quranic text (click to enlarge)
3. **Word-by-Word Table**:
   - Arabic word
   - Transliteration (pronunciation)
   - English translation
4. **Audio Player**: Click ▶️ to hear beautiful recitation
5. **Tafsir Section**: Scholarly interpretations (toggleable)

#### **Word-by-Word Analysis**

Each verse is broken down into individual words showing:

- **Arabic**: The original word in Uthmani script
- **Transliteration**: How to pronounce it in English letters
- **Translation**: What the word means in English

**Example:**
| Arabic | Transliteration | Translation |
|--------|-----------------|-------------|
| بِسْمِ | bismi | In (the) name |
| اللَّهِ | allahi | (of) Allah |
| الرَّحْمَٰنِ | ar-rahmani | the Most Gracious |

## 🛠️ **Troubleshooting**

### **Common Issues & Solutions**

#### **Audio Not Playing**

- Check your internet connection
- Ensure browser audio is enabled
- Try refreshing the page
- Some verses may not have audio available

#### **Tafsir Not Loading**

- Verify internet connection
- Try switching to a different tafsir source
- Refresh the page
- Check if the selected language/tafsir combination is available

#### **Navigation Issues**

- Use the search box for direct verse access
- Check your verse reference format (e.g., "2-255")
- Clear browser cache if needed

#### **Mobile Display Issues**

- Rotate to portrait mode for best experience
- Zoom out if content appears too large
- Clear browser cache
- Update your mobile browser

### **Performance Tips**

- **Close Unused Tabs**: Free up browser memory
- **Strong WiFi**: Better for audio streaming
- **Latest Browser**: Update for best performance
- **Clear Cache**: If experiencing slowdowns
